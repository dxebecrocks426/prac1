# GoDark DEX Blockchain Handbook

**Version:** 1.0

## Purpose

This handbook provides a comprehensive introduction to blockchain engineering subdivisions and explains how GoDark DEX fits into the broader blockchain ecosystem. It serves as foundational knowledge for all bootcamp participants, regardless of their specific assignment.

## Target Audience

- **~50 bootcamp participants** with varying blockchain experience levels
- **3 participants** with decent blockchain experience (can serve as mentors)

## Learning Path

### For Beginners
1. Start with blockchain subdivisions (01)
2. Understand Solana fundamentals (03)
3. Learn perpetual futures basics (04)
4. Review GoDark's role (02)
5. Study component overview (05)
6. Learn development workflow (06)
7. Practice testing strategies (07)
8. Review common patterns (08)
9. Study security practices (09)

### For Experienced Developers
1. Quick review of subdivisions (01)
2. Deep dive into GoDark architecture (02, 10)
3. Solana-specific patterns (03, 08)
4. Component integration points (05, 11)
5. Review security best practices (09)
6. Reference tooling and resources (12, 13, 14)
